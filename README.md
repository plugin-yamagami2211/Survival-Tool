# Survival-Tool
サバイバルを遊びやすくするかもしれないプラグインです。  

# 機能
## home機能
設定した任意の場所のTPすることができます。
どこのディメンションでも設定可能です。

## リスポーンにTPする機能
ベットなどのリスポーンポイントにTPすることができます。

#直前の死亡したポイントにTPする機能
直前の死亡したポイントにTPします。

## コマンド
/st : プラグインのヘルプコマンド  
/st reload : Configをリロードする  
/home : 設定した場所にTPする。  
/home set: 現在座標を記録する(home用)  
/respawn : リスポーンポイントにTPする。
